JavaScript

Javascript (JS) is a scripting languages, primarily used on the Web. It is used to enhance HTML pages and is commonly found embedded in HTML code. JavaScript is an interpreted language. Thus, it doesn't need to be compiled. JavaScript renders web pages in an interactive and dynamic fashion. This allowing the pages to react to events, exhibit special effects, accept variable text, validate data, create cookies, detect a userâ€™s browser, etc.

Operators

An operator is a special character (or characters) that indicates an action to be performed. +, -, *, /, and % are operators.

Variables

Variables are containers for storing data values. All JavaScript variables must be identified with unique names. These unique names are called identifiers. Identifiers can be short names (like x and y) or more descriptive names (age, sum, totalVolume).

Naming Conventions

Naming conventions are style guidelines for programming. They improve code readability and make code maintenance easier. It is also helpful when referring back to old code, or for team members, to know what the variables are for.

Functions

A method is a function which is a property of an object. There are two kind of methods: Instance Methods which are built-in tasks performed by an object instance, or Static Methods which are tasks that can be performed without the need of an object instance.

Methods

A function which is a property of an object. There are two kind of methods: Instance Methods which are built-in tasks performed by an object instance, or Static Methods which are tasks that can be performed without the need of an object instance.

Arguments

A value (primitive or object) passed as input to a function.

Parameters

A named variable passed into a function. Parameter variables are used to import arguments into functions.

Return

The return statement ends function execution and specifies a value to be returned to the function caller.

Chaining Methods

Two different methods being executed at the same time

Strings

A series of letters, numbers, and even punctuation within a pair of quotes.

Booleans

A logical data type that can have only the values true or false

Undefined

A variable without a value attached to it

NaN

Not a number. It is the returned value when Math functions fail (Math.sqrt(-1)) or when a function trying to parse a number fails (parseInt("blabla")).

Escape

This function computes a new string in which certain characters have been replaced by a hexadecimal sequence

Alert

This method displays a dialog with the optional specified content and an OK button.

Comments

Interface represents textual notations within markup; although it is generally not visually shown, such comments are available to be read in the source view

jQuery

A JavaScript Library that focuses on simplifying DOM manipulation, AJAX calls, and Event handling

Attributes

This extends a tag, changing tag behavior or providing metadata. It always has the form name=value.

This

Is determined by how a function is called.
